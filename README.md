# Crossword_CSP

Solving crosswords as a constraint satisfaction problem. 5 crossword of different sizes are included.

Smallest puzzle example:

['#', '#', '#', '#', '#', '#']<br>
['#', '_', '_', '_', '_', '#']<br>
['#', '_', '_', '_', '#', '#']<br>
['#', '_', '_', '_', '_', '#']<br>
['#', '#', '#', '#', '#', '#']

['need', 'boat', 'ban', 'art', 'ore', 'ate']

Result:

['#', '#', '#', '#', '#', '#']<br>
['#', 'b', 'o', 'a', 't', '#']<br>
['#', 'a', 'r', 't', '#', '#']<br>
['#', 'n', 'e', 'e', 'd', '#']<br>
['#', '#', '#', '#', '#', '#']
